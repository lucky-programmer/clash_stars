# clash_stars
python based clash stars made by grade 7 and 5


# How to play: 

## PLAYER ONE (LEFT PLAYER)
Move: W,A,S,D  
Shoot: SHIFT 

## PLAYER TWO (RIGHT PLAYER)
Move: DOWN ARROW, UP ARROW, RIGHT ARROW, LEFT ARROW 
Shoot: SPACE BAR

## Etc
RESTART: L
